## Installation:
1. Check the file `~/.config/kitty/kitty.conf` for the presence of the string `include current-theme.conf`
2. Add it if it's not there.
3. Copy file to `~/.config/kitty/`
4. Rename to `current-theme.conf` and overwrite it
5. Done!
